# AI Ambulance Priority Traffic Control System

This project implements an intelligent traffic signal system using:

- YOLOv8 object detection
- Arduino based signal control
- Real-time camera vehicle density estimation

## Features

✔ Ambulance priority signal  
✔ Density based green signal  
✔ No vehicle → both red  
✔ Two lane prototype  

## Run

pip install -r requirements.txt  
python python/detect_signal.py
